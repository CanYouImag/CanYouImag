# 👋 关于在下

我是*三清道人*，是一个正在考虑研究深度学习方向的，喜欢玄学领域的普通人。

哪都通快递…哦不，是代码搬运工。

在这个喧嚣的红尘里，我只想守着我那一亩三分地（服务器）。

入了尘世雕琢，才得出世般优越。

别和我扯什么“八奇技”，学不会。

风后奇门我不懂，玄机百炼也不会，炁体源流更是一窍不通，

但 Git 的门我还是能推开的。

看相的上左边，算命的去右边，拜师学道的出门左转去龙虎山。

闲杂人等一律退散，贫道正在闭关修炼（摸鱼）中。
